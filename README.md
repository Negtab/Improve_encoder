# 📘 Инструкция по использованию программы
## Шаг 1. Введите начальное состояние ключа
Введите последовательно двадцать восемь единиц или нулей

## Шаг 2. Открыть файл 
Нажмите [Открыть файл]: программа начнет генерацию ключа и шифрование файла.   
Прогресс работы программы будет отображаться ввиде ползунка

## Шаг 4. Сохраните результат
Нажмите [Сохранить результат]: результат сохранится в выбранный вами файл
